# To-Do List with Local Storage

A simple To-Do List web application built using HTML, CSS, and JavaScript.  
This app allows users to add, complete, and delete tasks while saving them in the browser using Local Storage.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks remain saved after page refresh
- Clean and simple user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

## How It Works

1. Enter a task in the input field.
2. Click the **Add Task** button.
3. Click on a task to mark it as completed.
4. Click **Delete** to remove a task.
5. All tasks are saved in the browser using Local Storage.
